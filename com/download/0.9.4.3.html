Hypertable 0.9.4.3 Binary Packages

This page contains binary packages for the Hypertable 0.9.4.3 release. Each package includes all of the dependent libraries, which is how we can provide a single set of generic packages for the Linux platform. The only requirement is that the Linux platform must be based on glibc 2.4 or higher. See the Hypertable Manual for installation instructions. Older Hypertable releases can be found at Hypertable Binary Package Archive

Release Notes

Note to Hadoop 0.21 Users: The Hypertable jar files in this release were built against Hadoop 0.20.2. To run this release on top of Hadoop 0.21, remove the Hadoop 0.20.2 jar files from the distribution and replace them with the Hadoop 0.21 jar files and remove the Hypertable jar files an replace them with the Hypertable jar files for Hadoop 0.21.
