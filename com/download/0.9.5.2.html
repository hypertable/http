Hypertable 0.9.5.2 Binary Packages

This page contains binary packages for the Hypertable 0.9.5.2 release. Each package includes all of the dependent libraries, which is how we can provide a single set of generic packages for the Linux platform. The only requirement is that the Linux platform must be based on glibc 2.4 or higher. See the Hypertable Manual for installation instructions. Older Hypertable releases can be found at Hypertable Binary Package Archive
