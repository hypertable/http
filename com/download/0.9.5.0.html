Hypertable 0.9.5.0 Binary Packages

This page contains binary packages for the Hypertable 0.9.5.0 release. Each package includes all of the dependent libraries, which is how we can provide a single set of generic packages for the Linux platform. The only requirement is that the Linux platform must be based on glibc 2.4 or higher. We've provided OS specific builds for Linux systems in which the generic packages are incompatible (see Linux OS Specific below). If the generic packages don't work for you platform, please post to the Hypertable Users mailing list and we'll build you a packages specific to your OS. See the Hypertable Manual for installation instructions. Older Hypertable releases can be found at Hypertable Binary Package Archive
